Raw HTML — copy this into your email client

▸ How to install in Gmail (Google Workspace)
  1. Open this file in your browser (you're doing that now).
  2. Click inside the textarea above — it will select all automatically.
  3. Copy it (Ctrl+C / Cmd+C).
  4. In Gmail, go to Settings → See all settings → General.
  5. Scroll down to Signature and click Create new.
  6. Name it PartLister.
  7. Click the <> (Insert HTML) icon in the signature editor toolbar.
  8. Delete any existing content in the HTML box, then paste your copied HTML.
  9. Click OK, then Save Changes at the bottom of the page.
  10. Set it as the default signature for New emails and Replies.
⚠️ Gmail strips the <style> block, so the mobile responsive rules won't apply in Gmail. The signature still looks great — the inline styles handle desktop perfectly. iPhone and Android Gmail apps do support mobile layout.
▸ How to install in Outlook (Windows)
  1. Open Notepad and paste the raw HTML from the textarea above.
  2. Save the file as partlister-signature.htm (choose "All Files" as the type, not .txt).
  3. In Outlook, go to File → Options → Mail → Signatures → New.
  4. Name it PartLister.
  5. In the signature editor, click Insert → Attach File — this won't work directly.
  6. Instead: navigate to %APPDATA%\Microsoft\Signatures\ in File Explorer.
  7. Paste your partlister-signature.htm file there.
  8. Back in Outlook Signatures, your new signature should appear — select it and click OK.
  9. Set as default for New messages and Replies/forwards.
⚠️ Outlook desktop uses Word's rendering engine. The VML conditional code in this signature ensures the CTA button renders with rounded corners in Outlook 2016–2021 and Microsoft 365.
▸ How to install in Apple Mail
  1. Open Apple Mail → Mail → Preferences → Signatures.
  2. Select your email account, click + to create a new signature.
  3. Name it PartLister.
  4. Quit Apple Mail completely (Cmd+Q).
  5. In Finder, go to ~/Library/Mail/V10/MailData/Signatures/ (the number may vary).
  6. Find the .mailsignature file that was just created — open it in a text editor.
  7. Replace everything after the headers (below the blank line) with the raw HTML from the textarea above.
  8. Save the file, then lock it (Get Info → tick Locked) so Mail doesn't overwrite it.
  9. Reopen Apple Mail and select your PartLister signature.
